# RainedUtilities
This is a set of utilities for the [Rained level editor](https://github.com/pkhead/rained).

## Installation

Merge the scripts folder included in this repository with the one in your Rained folder.

## Included utilities

### Tile Search AutoTile

Search for tiles compatible with the geometry of a specific tile in your level.

### Lua Geometry Export

Export the geometry file via Lua instead of Drizzle. You can run this via the menu option File > Export Geometry (lua).

Note: The geometry is not gauranteed to be exactly the same as what Drizzle would produce. Notably, the light angle will not be set.
